## ubuntu安装及配置nginx

### 安装
    sudo apt install curl gnupg2 ca-certificates lsb-release ubuntu-keyring

    sudo apt update

    sudo apt install nginx

### 编译代码方式安装
- 包地址：https://nginx.org/en/download.html
- 编译代码说明：https://nginx.org/en/docs/configure.html
